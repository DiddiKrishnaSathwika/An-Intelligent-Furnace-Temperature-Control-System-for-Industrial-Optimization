# An-Intelligent-Furnace-Temperature-Control-System-for-Industrial-Optimization
# RL Temperature Prediction

This project uses Reinforcement Learning to predict industrial furnace temperature from time-series data.

## Tech Stack
- Python
- PyTorch
- NumPy, Pandas
- Matplotlib

## Features
- Custom RL environment for time-series prediction
- Policy and Value network implementation
- Evaluation using MSE and MAE
- Visualization of predictions

## How to Run
1. Download dataset (eaf_temp.csv)
2. Place it in project folder
3. Run:
   python train_rl_model.py
